    Como votar usando o Sistema de Eleições

    1. Para acessar a cabina de votação, utilize suas credenciais do TJPR.

    2. A página inicial contém as instruções, clique no botão Iniciar

    3. Escolha os candidatos de acordo com a questão e se a eleição possuir mais de uma questão, então clique no botão Próxima Questão.

    4. Para a última questão, faça suas escolhas e clique no botão Próximo passo.

    5. Neste momento, suas escolhas são criptografadas em seu navegador para que, posteriormente, possam ser enviadas para contabilização.

    6. Nesta etapa é possível revisar as escolhas de voto. Caso esteja tudo correto, clique no botão Enviar este Voto!.

      • Se desejar alterar o voto, basta clicar no botão Editar e a tela retornará para a cédula de votação com todos os candidatos disponíveis para nova escolha.

    7. Clique no botão Depositar Cédula para depositar sua cédula na urna. Se não deseja depositar a cédula, clique no botão Cancelar.

      • Ao depositar a cédula na urna eletrônica o voto será contabilizado para a referida eleição.

    8. Parabéns, seu voto foi depositado com sucesso.
